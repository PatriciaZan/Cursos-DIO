## Projects

There were 6 projects in this course;

- Mario Kart <br>
  - A Revision on JS modules
- Carrinho de Compras <br>
  - Understanding Node Modules
  - Revision
- QRCode <br>
  - Using many modules
  - Packages manager
- Gerenciador Podcasts <br>
  - Raw project NodeJS
  - Typescript
  - Controllers | Models | Repositories | Routes | Services | Utils;
- Formula1 with Fastify <br>
  - Creating a full API with Fastify;
- Champions League ExpressJS <br>
  - A RestFull API for listing players, GET, PUT, DELETE and clubs, GET.
